## Controls

Escape : escape menu

Left arrow, A, Q : Left

Up arrow, W, Z : Up (use to enter doors)

Right arrow, D : Right

Down arrow, S : Down (also regenerates more energy, and allows characters to go through certain platforms)

Space Key : Jump
